## 📅 Week 1: Shopping Dataset Analysis

### Project: Shopping Dataset — EDA Project

**Location:** `week1/shopping-analysis/`

Exploratory data analysis, cleaning, and feature engineering on an e-commerce product dataset (Myntra fashion & lifestyle listings — 1,000 products across 97 categories).

**What's Included:**
- Data loading and exploration
- Data cleaning and preprocessing
- Feature engineering (price difference, popularity score, gender extraction)
- Univariate, bivariate, and category-level analysis
- Visualizations (histograms, bar charts, boxplots)
- Key insights and business implications

**Tech Stack:**
- Python 3
- pandas, numpy
- matplotlib, seaborn
- Jupyter Notebook

**To Run:**
```bash
cd week1/shopping-analysis/
pip install -r requirements.txt
jupyter notebook notebook/analysis.ipynb
```

## 📊 Key Findings (Week 1)

- **Discounting Dominance:** 90%+ of products have active discounts
- **Unrated Products:** ~11% of inventory has zero ratings (new/low-visibility items)
- **Category Concentration:** Top 5 categories account for majority of listings
- **Weak Discount-Rating Correlation:** Deep discounts don't guarantee higher ratings
- **Category-Specific Pricing:** Price ranges vary significantly by category
